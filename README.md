# linux-fundamentals-lab
Hands-on Linux practice based on real lab exercises, focused on command-line skills and security use cases. Includes writeups, screenshots, and practical problem-solving.
 

---
This repository documents my hands-on Linux learning journey using practical lab exercises. I am following structured learning from a cybersecurity-focused resource and completing all end-of-chapter exercises with real execution.

Instead of theory, this repo focuses on:

- Command-line usage
- Practical labs
- Real outputs
- Problem-solving approach

Goal: Build strong Linux fundamentals.
